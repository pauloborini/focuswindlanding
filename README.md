# FocusWind Landing

Landing page premium do FocusWind, com suporte a PT/EN, páginas legais em MD e
uma página dedicada de downloads.

## Páginas

- `index.html`: Landing principal com planos e chamadas para download
- `download.html`: Página de download com 4 plataformas (macOS, Windows, iOS, Android)
- `termos-de-uso.html`: Termos de Uso com conteúdo carregado de MD
- `politica-de-privacidade.html`: Política de Privacidade com conteúdo carregado de MD

## Estrutura do Projeto

- `style.css`: Estilos globais e componentes visuais
- `script.js`: I18n, renderização de MD e interações
- `docs/`: Conteúdo legal em PT e EN

## Como Executar

Basta abrir `index.html` em qualquer navegador moderno.

## Conteúdo Legal

Os documentos legais são carregados de:

- `docs/terms_pt.md` / `docs/terms_en.md`
- `docs/privacy_pt.md` / `docs/privacy_en.md`
